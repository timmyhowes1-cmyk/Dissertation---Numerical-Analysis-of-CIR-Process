## This repository contains all code used for the generation of results in my MSc dissertation paper, which is also included in this repository. 

Explicit_Schemes.py contains function definitions to generate the CIR process using explicit numerical approximation schemes.

Implicit_Schemes.py contains function definitions to generate the CIR process using implicit numerical approximation schemes.

Useful_Functions.py is a bunch of helper function definitions.

Strong_Error_Analysis.py contains the code to generate strong error analysis and plots via Monte Carlo simulation.

Heston_Analysis.py contains the code to generate weak error analysis and plots via Monte Carlo simulation under the vanilla call option Heston pricing model.

Bond Pricing.py contains the code to generate weak error analysis and plots via Monte Carlo simulation under the ZCB pricing model under CIR short rate dynamics.

Parameter_Analysis.py contains the code to generate convergence analysis under different Feller ratio scenarios.

Prob_of_0.py contains the code to analyse the probability of schemes taking value 0.

Prob_of_neg.py contains the code to analyse the probability of schemes taking negative values.

Plot Schemes.py contains code to generate plots of the CIR process approximated by two numerical schemes.

Moments Analysis.py contains the code to calculate MC errors for two approximation schemes.
